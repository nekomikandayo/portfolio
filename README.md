### 📘 Reading Habit（習慣化読書アプリ）
**短時間の読書を「振り返り」と「習慣化」につなげる読書支援Webアプリ**

- 忙しい日常でも読書を継続できるよう設計
- UI・画面遷移・データ構造を自分で設計
- 読書時間の計測と、気づきの即時アウトプットで定着を支援
- Firebaseと連携し、認証・データ保存を実装

🌐 **Live Demo**  
https://reading-habit-app.web.app  

💻 **Source Code**  
https://github.com/nekomikandayo/reading-habit-app
