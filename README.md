- 💻 [Source Code](https://github.com/nekomikandayo/calculator-app)

### 🍅 ポモドーロタイマー / 猫の屈伸

猫と一緒に集中できる学習支援アプリ。画面遷移やタイマー状態の変化を事前に設計し、  
UIの動作フローを整理してから実装することで修正コストを削減しました。  
LocalStorageを用いた学習記録の永続化・統計ページも実装しています。

- 🌐 [Demo](https://nekomikandayo.github.io/cat-pomodoro/)
